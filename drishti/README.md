# DRISHTI

**AI-Powered Crime Intelligence Platform**

Built for the KSP Datathon 2026, organized by Karnataka State Police and Hack2skill, with Catalyst by Zoho as tech/hosting partner.

## Overview

DRISHTI centralizes crime records, analytics, and investigation support for law enforcement, offering:

- Conversational crime assistant
- Crime hotspot & trend analysis
- Criminal network visualization
- AI-driven case intelligence
- Automated reports & voice access

## Tech Stack

**Frontend:** React.js, Chart.js, Leaflet.js, Cytoscape.js
**Backend:** FastAPI, Zoho Catalyst Functions
**Data & AI:** Catalyst Data Store, Catalyst QuickML, Pandas, NumPy, Scikit-learn
**Platform Services:** Catalyst Authentication, Zia Services, SmartBrowz
**Deployment:** Zoho Catalyst Cloud
